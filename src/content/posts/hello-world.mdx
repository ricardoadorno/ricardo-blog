### Multi-language Code Examples

JavaScript:
```javascript
const greeting = (name) => {
  console.log(`Hello, ${name}!`);
};

greeting('World');
```

TypeScript:
```typescript
const greeting = (name: string): void => {
  console.log(`Hello, ${name}!`);
};

greeting('World');
```

Python:
```python
def greeting(name: str) -> None:
    print(f"Hello, {name}!")

greeting("World")
```
